# Menu

Drop your menu PDF in here with this exact filename:

**`brothers-taquizas-menu.pdf`**

The "View Full Menu" buttons on the homepage point to `/menu/brothers-taquizas-menu.pdf`.

## Current source

The current menu PDF lives on Shopify here:
https://cdn.shopify.com/s/files/1/0985/6250/7118/files/Brothers_Taquizas_Menu.pdf?v=1774363236

Right-click → "Save link as…" → save it into this folder as `brothers-taquizas-menu.pdf`.

## When the menu changes

You only need to update one file. Just:
1. Replace `brothers-taquizas-menu.pdf` in this folder with the new one (same filename)
2. Commit and push
3. Vercel redeploys — the "View Full Menu" buttons now link to the new PDF automatically
